Markup first, style second

  • Mark things up semantically and the styles should follow.
  • In general, text should not live naked inside a <div> unless you're doing something 1%-ish.

Occupy stylesheet

  • The default styles should work for the 99% of elements on your page.
  • If an element is truly part of the 1%, wearing a monocle and standing on a yacht, use modifier classes and page-specific styles.

Blocks that fit together

  • Padding (top, right, left) happens on wrappers.
  • Margin (bottom) happens on block-level content.
