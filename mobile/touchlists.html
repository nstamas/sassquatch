Right now, the touchList class is applying some overrides to the way font-size, margin and padding are defined in sectionLists and dividedLists

Basic TouchList

This is an example of the most basic touchlist.

  • Item 1

  • Item 2

  • Item 3

     
<%-- JSP --%>
<div class="doc-box">
    <ul class="touchList">
        <li:item label="Item 1" action="url" />
        <li:item label="Item 2" action="url" />
        <li:item label="Item 3" action="url" />
    </ul>
</div>

Attendee Item TouchList

Initially designed for the Attendee/RSVP list on event details, we can use this pattern whenever we need to display member data in a list.

  • Captain Spock (+ 5)

    Vulcan
    Good to see you

  • Leonard McCoy

    "Bones"
    Good to see you

  • Lieutenant Uhura

    Good to see you

  • Evil Kirk

    Good to see you 
<%-- JSP --%>

<li:attendee 
	attendee="${rsvp}" 
	listing="${listing}" 
	rsvp_type="${rsvp_type}" 
	id="rsvp_${rsvp.member.memberId }" 
	json='{	"memberId": ... }'
/>

<!-- HTML -->

<div class="doc-box">
  <ul class="touchList">
	<li class="touchList-item--attendee">
		<div class="ff-box">
		  <div class="ff-box-flex">
			<a href="${member_url}" class="figureset j-btn touchList-item--attendee-figureset">
			  
			  <!-- image -->
			  <div class="figureset-figure avatar-m pin-top">
				<img src="assets/img/spock.jpg" />
			  </div>
			  <!-- avatar and info -->
			  <div class="figureset-description">
				<h4 class="touchList-item--attendee-label">
				  Captain Spock
				  <span class="touchList-item--attendee-label-guest-count">(+ 5)</span>
				</h4>
				<h6 class="touchList-item--attendee-role">Vulcan</h6>
			  </div>
			</a>
		  </div>

		  <!-- secondary actions -->
		  <div class="ff-box-fix touchList-item--attendee-secondary-actions">
			<a class="button small niceToSeeYou primary" data-memberId="${attendee.member.id}" href="#">
			  <span>Good to see you</span>
			</a>
		  </div>
		</div>
	</li>
	<li class="touchList-item--attendee">
		<div class="ff-box">
			<div class="ff-box-flex">
				<a href="${member_url}" class="figureset j-btn touchList-item--attendee-figureset">
					<div class="figureset-figure avatar-m pin-top">

					</div>

					 <div class="figureset-description">
						<h4 class="touchList-item--attendee-label">
						  Leonard McCoy
						</h4>
						<h6 class="touchList-item--attendee-role">"Bones"</h6>
					</div>
				</a>
			</div>
		
			<div class="ff-box-fix touchList-item--attendee-secondary-actions">
