OOCSS Grids are used to break any space into fractions, they can be nested and stacked. A unit can contain another line or it can contain other objects directly. Grids control width, content controls height.

.line is a row. .unit is a column.

DEPRECATED: .gutter-left, .gutter-right (resulted in uneven width of lastUnit)

See also: https://github.com/stubbornella/oocss/wiki/Grids

Box and sections

.doc-box is the main box of the page. 99% of the time, there should be only one.

Optionally, your box can be divided into columns by using the standard grid, if you so desire. Do not use gutters here.

Next, a "section" of content is created by adding .doc-content . Simple boxes may have only one section.
