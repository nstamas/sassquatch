Buttons

As a rule, buttons are for actions, not navigation. This style applies to 

<input type="submit" />, <button>
, and anything with the class .button.

The default is a simple gray button. There is also red .primary class for "do this one thing" actions.

For side-by-side inputs and buttons, you'll probably want something a little smaller than the default button. For this, you can use the .small class.

Default stacked forms

To group related label & input pairs, put them in a fieldset. Use OOCSS grids to divide into columns if needed.

Where are you? 
 <form>
	<label for="name">Name</label>
	<input id="name" type="text">

	<label for="color">Favorite color</label>
	<select name="color">
		<option>Midnight Saffron</option>
		<option>Seabreeze Brown</option>
		<option>Fragrant Forest Ochre</option>
	</select>

	<fieldset class="line-gutters">
		<legend>Where are you?</legend>
		<div class="unit size1of2">
			<label for="city">City</label>
			<input type="text" id="city">
		</div>
		<div class="lastUnit">
			<label for="state">State</label>
			<select>
				<option>New York</option>
				<option>New Jersey</option>
				<option>New Mexico</option>
				<option>New Oregon</option>
			</select>
		</div>
	</fieldset>

	<label for="unicorn">Describe your unicorn</label>
	<textarea name="unicorn"></textarea>
</form>

Checkboxes and radio sets

Checkbox and radio inputs are placed inside the <label> elements for larger click targets.

The default label size applies here, but you can add a .small modifier to the labels if you prefer something smaller

<input type="checkbox" />, <input type="radio" />

I'm interested in... 
 <form>
	<fieldset>
		<legend>I'm interested in...</legend>
		<label class="small"><input type="checkbox" /> Cats</label>
		<label class="small"><input type="checkbox" checked="checked" /> Honey Badgers</label>
		<label class="small"><input type="checkbox" /> Quails</label>
		<label class="small"><input type="checkbox" checked="checked" /> Goats</label>
		<label class="small"><input type="checkbox"  checked="checked" /> Red Pandas</label>
	</fieldset>
</form>

Form field sizes

There are width helpers defined in layout.scss that can assist in changing the default width of fields. You can ajust the width by thirds and quarters of the width of the parent container.

.span-25, .span-30, .span-50, .span-60, .span-75

.span-25 .span-30 .span-50 .span-60 .span-75 .span-100 Default (width: auto;)

Required/Error states

input.error,label.required

Required field labels can be marked as required with the .required class. Error states are applied with an .error class on the input element.

You are not Keith Hernandez.

<form class="span-60">
	<label for="address">Address</label>
	<input id="address" value="123 Fake St." type="text" />

	<label class="required" for="name">Name</label>
	<input class="error" id="name" value="Keith Hernandez" type="text" />
	<p class="italic">You are not Keith Hernandez.</p>

	<label for="address">Profession</label>
	<input id="address" value="Blimp Captain" type="text" />
</form>
