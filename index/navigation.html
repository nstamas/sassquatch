Pagination

Pagination is for paginating. Use the rel attributes of the "Previous" and "Next" anchors to indicate prev or next. Not only is it good markup, we use it as a styling hook for adding the nifty arrows.

.nav-pagination

  • ←
  • 1
  • 2
  • 8
  • 9
  • → 
 <ul class="nav-pagination">
	<!-- ideally, you'd use an icon font for the arrows -->
	<li class="nav-pageitem">
		<a rel="prev" class="nav-prev" href="#">&larr;</a>
	</li>
	<li class="nav-pageitem">
		<a href="#">1</a>
	</li>
	<li class="nav-pageitem selected">
		<a href="#">2</a>
	</li>
	<!-- snip -->
	<li class="nav-pageitem">
		<a href="#">8</a>
	</li>
	<li class="nav-pageitem">
		<a href="#">9</a>
	</li>
	<li class="nav-pageitem">
		<a rel="next" class="nav-next" href="#">&rarr;</a>
	</li>
</ul>

Button groups (filtering)

Buttons joined in groups, presented as a list of options. Most useful for filtering/sorting lists, but we may be able to use this for other things.

.nav-buttons

Sort by:
  • Date Joined
  • Size
  • Name
  • Role
<ul class="nav-buttons">
	<li><a >Date Joined</a></li>
	<li><a class="selected">Size</a></li>
	<li><a >Name</a></li>
	<li><a >Role</a></li>
</ul>
