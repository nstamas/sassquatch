.paddedList

Simple .paddedList

  • List items are divided...

  • This doesn't need any special "first" or "last" classes

  • You can subdivide these list items into columns with the standard grid, if you like

Data tables

Data tables should be used for tabular data, of course. In most cases on Meetup, we omit the <thead> and <caption>. If you do however, need table headers, you should be using <thead> instead of styling a table body row to look like table headers.
